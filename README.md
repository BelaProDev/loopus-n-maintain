## Quick Start

1. Clone and install:
```bash
npm install
```

2. Configure environment:
```env
SMTP_HOSTNAME=
SMTP_USERNAME=
SMTP_PASSWORD=
CONTACT_FORM_RECIPIENTS=
```

3. Run development server:
```bash
npm run dev
```
