## Quick Start 🌷

1. Clone the repo
2. Install dependencies:
```bash
npm install
```

3. Set up environment variables:
```env
VITE_FAUNA_SECRET_KEY=your-fauna-key
VITE_DROPBOX_ACCESS_TOKEN=your-dropbox-token
```

4. Start development server:
```bash
npm run dev
```

5. Visit `localhost:5173` 😊
